# Math Tower - Çalıştırma Kılavuzu

Bu proje bir **Expo** projesidir. Ana bilgisayarına geçtiğinde şu adımları izleyerek oyunu çalıştırabilirsin:

## Kurulum Adımları

1.  **Node.js Kurulu Olduğundan Emin Ol:** Yoksa [nodejs.org](https://nodejs.org/) adresinden kur.
2.  **Dosyaların Olduğu Dizine Git:** Terminal aç ve proje klasörüne gir.
3.  **Bağımlılıkları Yükle:**
    ```bash
    npm install
    ```
4.  **Uygulamayı Başlat:**
    ```bash
    npx expo start
    ```
5.  **Telefonda Dene:** Telefonuna **"Expo Go"** uygulamasını (App Store/Play Store) indir ve ekrandaki QR kodu tarat.

## Teknik Notlar
- Gerekli tüm paketler (`zustand`, `lucide-react-native`, `reanimated`) `package.json` dosyasına eklenmiştir.
- TypeScript ayarları senior düzeyinde `strict` modda yapılmıştır.
